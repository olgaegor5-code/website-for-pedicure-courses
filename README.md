# Педикюр.школа — лендинг

Статический одностраничный сайт: приветственный блок, программы обучения, FAQ и форма обратной связи (валидация в браузере).

## Структура

| Файл | Назначение |
|------|------------|
| `index.html` | Разметка и контент |
| `styles.css` | Стили, светлая и тёмная тема |
| `script.js` | Меню, тема, приветствие, FAQ, форма, год в подвале |

## Как открыть

- Откройте `index.html` в браузере двойным щелчком, **или**
- Из корня проекта: `npx --yes serve .` и перейдите по адресу, который покажет терминал.

## Настройка под себя

1. В блоке «Связаться» замените `hello@example.com` и ссылки на Telegram/WhatsApp.
2. При необходимости подключите отправку формы на сервер (например, Formspree, Netlify Forms) — сейчас после успешной проверки показывается сообщение об отправке.

## Лицензия

Используйте и изменяйте свободно для своих целей.
